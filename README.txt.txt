Project Name: Smart Parking Management System

Description:
This project is developed using Python Flask and MySQL to manage parking slots.

Features:
- Login system
- Add new parking slots
- Book and unbook slots
- View available and occupied slots

Technologies Used:
- Python
- Flask
- MySQL
- HTML

How to Run:
1. Install Python and MySQL
2. Install required libraries: flask, mysql-connector-python
3. Run app.py
4. Open http://127.0.0.1:5000
